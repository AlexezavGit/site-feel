## MHEI v3.0 формула (канонічна)
```
MHEI = 0.30×C + 0.20×B + 0.20×E + 0.15×S + 0.15×G

C = клінічне відновлення (remission rate, ΔSym, adherence)
B = бюджетна ефективність (avoided costs / program costs)
E = економічне повернення (RTW × Y - L / PC)
S = соціальна доступність (coverage, equity, stigma reduction)
G = governance (data quality, privacy, auditability, rights protection)
```
Поточний стан: 4.12 → Цільовий: 8.16 (шкала 0-10)
