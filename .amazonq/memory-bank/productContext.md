# Product Context

## Problem Statement
Світовий Банк інвестував $954M в охорону психічного здоров'я України (HEAL $500M + THRIVE $454M), але 0.28% клінічної потреби покрито, $954M частково заблоковано через відсутність цифрового зв'язку між гуманітарними даними та державною системою ESOZ. FEEL Again — middleware-ключ. Dashboard — інструмент переконання інвесторів та WB.

## Target Users
- Аналітики та менеджери Світового Банку (HEAL/THRIVE project teams)
- МОЗ України, НСЗУ — для обґрунтування цифровізації
- Міжнародні донори (USAID, UNICEF, WHO, EU) — для координації
- Інвестори в FEEL Again як технологічну компанію

## User Experience Goals
- Аналітик WB повинен за 2 хвилини зрозуміти масштаб кризи та роль FEEL Again
- Кожна цифра має бути верифікована та клікабельна (джерело)
- Дашборд має виглядати як офіційний аналітичний інструмент, а не стартап-презентація

## Key User Flows

### Flow 1: WB Analyst Review
1. Відкриває dashboard.feelagain.me
2. Бачить L1 Strategic — 6 crisis metrics (0.28%, 62.2M, $954M, 7.8yr, 0%, $5M/mo)
3. Переходить в L2 Finance або L2 Digital для деталей
4. Відкриває L3 Appendix для повної аналітики
5. Завантажує L4 Full Report або надсилає посилання колезі

### Flow 2: Investment Pitch
1. Презентація починається з L1 (crisis scale)
2. Перехід до L2 MHEI або L2 Operational для ROI
3. CTA: "Schedule Demo" або "Get Deck"

## Competitive / Prior Art Context
- OWID, IHME GHDx, World Bank Open Data — академічні, без інвестиційного наративу
- Відсутній будь-який дашборд, що поєднує MHPSS + macroeconomics + WB project tracking для України
- FEEL Again Dashboard — унікальний у своїй ніші
