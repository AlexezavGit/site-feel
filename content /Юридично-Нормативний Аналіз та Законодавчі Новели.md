# **Юридично-Нормативний Аналіз та Законодавчі Новели (Закон № 4223-IX)**

Документ містить безапеляційний критичний аудит Закону України № 4223-IX «Про систему охорони психічного здоров'я в Україні» та пропозиції щодо його реформування.

## **1\. Регуляторний Аудит Чинної Бази (Приховані Бар'єри)**

* **Матеріально-технічний бар'єр (Ценз кабінету):** Вимога відповідності кабінетів ДБН для ЗОЗ приватної практики штучно витісняє фахівців у тінь, ігноруючи реалії ринку, де 75% сесій проходять онлайн.  
* **Заборона вільного ціноутворення:** Вимога фіксованих тарифів при будь-якому державному співфінансуванні знищує рентабельність приватної практики.  
* **Дискримінація за статусом:** Використання високоінтенсивних протоколів (EMDR, КПТ, VR) штучно обмежено ЗОЗ та фахівцями з клінічним статусом. Сотні сертифікованих EMDR-практиків у цивільному секторі опиняються у напівлегальній зоні, якщо працюють із діагнозом ПТСР.

## **2\. Наслідки для Споживачів та Держави**

**Маскування послуг:** Приватні психологи змушені укладати договори на "інформаційні послуги" або "коучинг". Внаслідок цього пацієнт втрачає правовий статус споживача медико-психологічної допомоги і не захищений Законом «Про захист прав споживачів».  
**Інституційне марнотратство:** Закон де\-факто захищає застарілу інфраструктуру колишніх психдиспансерів, змушуючи НСЗУ фінансувати неефективні багаторазові стаціонарні цикли замість амбулаторного результату.

## **3\. Пропоновані Законодавчі Новели**

Для усунення макроекономічних ризиків необхідно інтегрувати компетентнісний принцип EuroPsy та модель фінансової децентралізації.

| Сфера регулювання | Пропонована Новела |
| :---- | :---- |
| **Доступ до практики (Демонополізація)** | Надання амбулаторної психотерапевтичної допомоги дозволяється суб'єктам (в т.ч. ФОП) на основі диплома Магістра 053 та чинної сертифікації за доказовим методом від профільної асоціації. Вимога медичної ліцензії ЗОЗ та "медичного кабінету" скасовується для приватної амбулаторної практики. |
| **Фінансування та Ціноутворення** | Держава гарантує суб'єктам приватної практики право на вільне ринкове ціноутворення. Дозволяється багатоджерельне фінансування (Multi-source Waterfall Billing). Граничні тарифи діють лише у випадку 100% покриття послуги з бюджету НСЗУ. |
| **Інтероперабельність (Digital Bus)** | Впровадження автоматизованого шлюзу між гуманітарними базами та ЕСОЗ. Використання деперсоналізованих проксі-даних для первинної верифікації без обов'язкової постановки на психіатричний облік (захист від стигми). |
| **Податкове стимулювання (Tax Relief)** | Звільнення приватних психологів від 5% єдиного податку ФОП, якщо понад 50% їх клієнтів мають верифіковане в ЕСОЗ клінічне покращення (позитивна динаміка за PHQ-9/PCL-5). |
